Introduction

HTML (HyperText Markup Language) gives structure to web content. Use semantic elements like <header>, <main>, <section>, and <footer> for clarity and accessibility. 

<h1>Title</h1>
<p>Paragraph text.</p>

Images & Media

Always include descriptive alt text for images.

<img src="cat.jpg" alt="A gray cat" width="320">

Tables

Tables are for tabular data. Use <thead> and <tbody>. 

<table>
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Siddhartha</td><td>22</td></tr>
</table>
